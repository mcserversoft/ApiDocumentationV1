MCSS API v13 (1.0.0)

Download OpenAPI specification:Download

Documentation for the official MC Server Soft API.

This for MCSS v13 and higher.
The API used in V12 can be found at https://documenter.getpostman.com/view/7749159/TzJsexSC

Enabling SSL is recommended even when the certificate is self signed!

Get server count

Get the total amount of servers.

filter (optional) : enum
0 = None
1 = Online
2 = Offline
3 = ByServerType

serverTypeId (required when using ByServerType) : guid
Guid of the server type.

Authorizations:
apikeyAuth
query Parameters
filter
string
Example: filter=Online
serverTypeId
string
Example: serverTypeId=f738fb40-223e-4010-bd07-af4caabdf3dd

Responses

Your local MCSS API server

https://localhost:25560/api/v1/servers/count

Response samples

  • 200
  • 401
Content type
application/json
{
  • "count": 5
}

Console

API calls that involve the console from a single server.

Get server console

Return the console of a specific server.

amountOfLines (required) : integer
Amount of lines you want to fetch, -1 to fetch everything.

takeFromBeginning (optional) : boolean
Specify if you want to start fetching from the oldest lines first.

reversed (optional) : boolean
Option to flip the line order.

Authorizations:
apikeyAuth
query Parameters
AmountOfLines
integer
Example: AmountOfLines=5
Reversed
boolean
Example: Reversed=true
takeFromBeginning
boolean
Example: takeFromBeginning=true

Create task

Create a new task.

task (required)
- name (required) : string
- enabled (optional) : boolean
- timing (required) : object
- job (required): object

Types of Timings (3)

fixedTime
- repeat: boolean
- timeSpan: TimeSpan

interval
- repeat: boolean
- interval: integer

timeless
// no parameters

Types of Jobs (3)

ServerAction
- action : ServerAction
// 0 or Stop
// 1 or Start
// 2 or Kill
// 3 or Restart

RunCommands
- commands: string[]

StartBackup
- backupIdentifier: guid

Authorizations:
apikeyAuth
Request Body schema: application/json
object

Responses

Your local MCSS API server

https://localhost:25560/api/v1/servers/e0ae4902-10ed-481f-aed0-fa1170ec779a/scheduler/tasks
